# SwarmAI Desktop

Tauri 2.0 desktop application for SwarmAI.

See the main [README](../README.md) (English) or [README.zh-CN.md](../README.zh-CN.md) (中文) for full documentation.

## Development

```bash
npm install
npm run tauri:dev
```

## Build

```bash
npm run build:all
```

Build output: `src-tauri/target/release/bundle/`
